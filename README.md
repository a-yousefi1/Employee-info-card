# Employee Info Card
This repository contains a small project called "Employee Info Card" that displays fictional information about employees in a company. The project is designed to showcase the details of 4 employees, and users can navigate between the employee information using buttons. To move to the next employee, simply click on the "Next" button, and to go back to the previous employee, click on the "Previous" button. If you want to select a random employee, click on the "Surprise Me" button.

## Technologies Used
This project is implemented using HTML, CSS, and JavaScript.

## Getting Started
To run the project locally, follow these steps:

1. Clone this repository to your local machine.
2. Open the project folder.
3. Open the index.html file in your web browser.
4. You will see the employee info card with the details of the first employee.
5. Use the "Next" and "Previous" buttons to navigate between employees.
6. Click on the "Surprise Me" button to select a random employee.
